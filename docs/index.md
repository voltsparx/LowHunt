## What LowHunt Is For

LowHunt is for public-data-only OSINT on authorized targets. It combines:

- username footprint scanning
- passive domain host collection
- public contact-email harvesting
- cross-signal correlation
- investigator-friendly reporting

## What Makes It Useful

- native C runtime
- multiple execution engines
- passive source fusion
- built-in noise reduction
- stored report bundles
- combined investigation mode
