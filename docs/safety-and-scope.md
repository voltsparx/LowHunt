# Safety and Scope

## Intended Use

LowHunt is for public-data-only OSINT on authorized targets.

## Allowed Use Cases

- defensive investigations
- exposure review
- account footprint checks
- passive domain intelligence
- case-building with publicly visible data

## Out of Scope

LowHunt is not intended for:

- bypassing authentication
- defeating protections
- credential attacks
- hidden active intrusion behavior
- unauthorized target analysis

## Operator Responsibility

You are responsible for:

- ensuring the target is in scope
- respecting local law and policy
- verifying whether findings are current and correctly attributed
- using report bundles carefully when sharing sensitive investigations

## Interpretation Guidance

Public OSINT findings can be:

- incomplete
- stale
- misleading
- shared across unrelated parties

LowHunt helps organize signals. It does not replace analyst judgment.
